# AR-Car

Augmented Reality Car Application - An interactive AR experience that brings virtual cars to life in real-world environments.

## Overview

AR-Car is an augmented reality application that enables users to visualize and interact with virtual 3D car models in their physical space using mobile devices. The app leverages cutting-edge AR technology to provide an immersive and engaging experience for car enthusiasts and buyers.

## Features

- **3D Car Visualization**: View realistic 3D car models in augmented reality
- **Real-time Interaction**: Rotate, scale, and position cars in your environment
- **Multiple Vehicle Options**: Browse different car models and configurations
- **Environmental Integration**: Cars adapt to lighting and surfaces in real-world scenes
- **Cross-Platform Support**: Works on both Android and iOS devices

## Tech Stack

- **Engine**: Unity 3D
- **AR Framework**: AR Foundation
- **AR Libraries**: ARCore (Android) / ARKit (iOS)
- **Language**: C#
- **Platform**: Mobile (Android & iOS)

## Installation & Setup

### Prerequisites

- Unity 2019.4 or later
- AR Foundation package
- ARCore (Android) or ARKit (iOS) SDK
- Compatible mobile device with AR support

### Build Instructions

1. Open the project in Unity
2. Install AR Foundation from the Package Manager
3. Set up your target platform (Android or iOS)
4. Configure build settings
5. Build and deploy to your device

## Usage

1. Launch the application on your mobile device
2. Allow camera and location permissions
3. Point your device at a flat surface
4. Place a 3D car model in your environment
5. Use touch gestures to rotate and scale the model

## Touch Controls

| Gesture | Action |
|---------|--------|
| **Tap** | Select/Place car |
| **Pinch** | Scale car size |
| **Rotate** | Two-finger rotation |
| **Drag** | Move car position |
